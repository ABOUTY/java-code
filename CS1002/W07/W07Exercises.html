You are provided with some starter code, including empty methods exercise1 to exercise7. Your task is to complete the bodies of these methods according to the instructions below.

Create a new folder called W07Exercises inside your CS1002 folder. Download the file W07Exercise1.java and move it into the new folder. Open the file in VS Code.

Exercises

Exercise 1

Inside method exercise1, add code to create an integer array of a size input by the user. It should then repeatedly prompt the user to input an integer to be stored in each element of the array. Finally, it should print out each element of the array on a separate line. The following example illustrates the format required by stacscheck:

Enter number of elements:
3
Enter element:
4
Enter element:
1
Enter element:
17
Elements:
4
1
17
 Run the program to check it works, and then run stacscheck:

stacscheck /cs/studres/CS1002/Exercises/W07/Tests/

At this stage, if your first exercise works correctly, you should see 5 tests passed.

Exercise 2

Many of this week's exercises require you to create an integer array based on user input. To avoid having to repeat this code in every case, create a method called getArray to do this. You can copy and paste relevant code from exercise 1 if you like.

Now complete method exercise2, which should do the same thing as exercise1, but calling your getArray method to create the array.

Test manually, then run stacscheck again and fix any errors in this exercise. At this stage you should see 6 tests passed.

Exercise 3

Continue in the same way to add methods for the other exercises. Run stacscheck after adding each one. Exercise 3 should call getArray, and then find the maximum value within it, by checking every element. It should print out the maximum value after the label Maximum value:

Test manually, then run stacscheck again and fix any errors in this exercise. At this stage you should see 8 tests passed.

Exercise 4

This should create an array using getArray, and then calculate the mean of the values within it. It should print out the mean value after the label Mean value:. Note that while the array elements are integers, the mean may not be an integer.

Test manually, then run stacscheck again and fix any errors in this exercise. At this stage you should see 9 tests passed.

Exercise 5

This should create an array using getArray, and then prompt the user to enter an integer value, with the prompt Enter element to search for:. It should then search through the array to see whether it contains that value. The method should print the index (i.e. position within the array) of the first occurence of the value, or -1 if the value is not found, after the label First occurrence at index:.

Test manually, then run stacscheck again and fix any errors in this exercise. At this stage you should see 13 tests passed.

Exercise 6

This exercise is the same as the previous exercise, except that it should search for the last occurrence of the given value, rather than the first. The result should be printed with the label Last occurrence at index:.

Test manually, then run stacscheck again and fix any errors in this exercise. At this stage you should see 14 tests passed.

Exercise 7

This exercise uses two-dimensional arrays, so you should not use getArray.

The method should create a 2-D integer array of size 3 x 3. It should then prompt the user to enter a value for each element, using a prompt specifying the particular indexes. For example, to input the value for the element with indexes [2][1]:

Enter element [2,1]:

It should then create a second array of the same size as the first, and fill in the elements such that the second array is the transpose of the first, i.e. the rows and columns are swapped. Finally, it should print out the second array after the label Transposed array:. For example, if the contents of the first array input by the user are: 

7 3 4
17 2 17
-7 0 17

then the output should be:

Transposed array:
7 17 -7
3 2 0
4 17 17

Test manually, then run stacscheck again and fix any errors in this exercise. At this stage you should see all tests passed.
